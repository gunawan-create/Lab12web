## Struktur folder project
```
lab11_php_oop/
 ├── .htaccess
 ├── config.php
 ├── index.php
 ├── class/
 │    ├── Database.php
 │    └── Form.php
 ├── module/
 │    └── home/
 │         └── index.php
 └── template/
      ├── header.php
      ├── footer.php
      └── sidebar.php

```
Keterangan:
- Folder class/ berisi library yang kita gunakan
- Folder module/ berisi halaman-halaman web
- Routing diarahkan ke folder module secara otomatis
